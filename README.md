# Jogo isKahoot

Este projeto foi realizado no âmbito da UC Programação Concorrente e Distribuída e implementa um jogo estilo Kahoot que usa um modelo servidor-cliente (um servidor central e vários clientes), pelo que existem várias situações de concorr
ência que foram tratadas usando mecanismos de sincronização e coordenação (como métodos synchronized no recurso partilhado e mecansimos de coordenação como barreiras e countdown latch modificados para incluir a funcionalidade de temporizador e cálculo de pontuação).
